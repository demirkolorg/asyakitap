## ADR Şablonu

Her ADR aşağıdaki yapıyı takip eder:

```markdown
# ADR-XXX: Başlık

## Durum
Önerildi | Kabul | Reddedildi | Değiştirildi | Kullanımdan Kaldırıldı

## Bağlam
Karar gerektiren durum veya problem.

## Karar
Alınan karar ve nedeni.

## Sonuçlar
Kararın olumlu ve olumsuz etkileri.
```
